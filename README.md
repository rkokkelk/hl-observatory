# HL Observatory #

## Installation ##

Run the following commond line:

`pip3 install flask`

## Usage ##

Run the following:

`python3 observatory.py`

This will start the server on your localhost port 5000

Then the API can be called using curl, e.g.

`curl http://localhost:5000?domain=google.com`
